## Блок A — Видение и стратегия

### Миссия

**music-dl** — кросс-платформенный инструмент для **личного архивирования** музыки из стримингов, на которые у пользователя уже есть подписка. Не замена подписке, а удобство: скачать, разметить, хранить локально.

### Путь к заработку (высокий уровень)

```
Open-source CLI          →  доверие, community, GitHub
        ↓
Больше провайдеров (VK…)  →  ценность продукта растёт
        ↓
GUI + сайт               →  аудитория «без терминала»
        ↓
Freemium Pro             →  первые деньги (лицензия GUI)
        ↓
Облако (бэкенд)          →  подписка, повторяющийся доход
        ↓
Mobile                   →  расширение Pro, APK с сайта
```

### Принципы монетизации

1. **Продаём удобство**, не «обход подписок».
2. **CLI остаётся бесплатным** — open source, MIT.
3. **Pro = GUI + очередь + sync + все провайдеры + поддержка**.
4. **Не проксируем музыку** через сервер — скачивание на устройстве пользователя.
5. **Disclaimer** на сайте и в приложении — личное использование.

### Целевые метрики (ориентир)

| Этап | Срок | Метрика |
|------|------|---------|
| Community | сейчас | 100+ ⭐ GitHub, первые issues |
| Продукт | +3 мес. | GUI + VK + сайт |
| Первый доход | +4–6 мес. | Boosty / первые Pro-лицензии |
| Стабильный доход | +12 мес. | 50+ Pro-подписчиков или 10–100 тыс. ₽/мес |

### E.3 Бэкенд (API)

**Когда нужен:**

| Сценарий | Бэкенд |
|----------|--------|
| CLI / десктоп, локальный токен | ❌ не нужен |
| Статический сайт + Releases | ❌ не нужен |
| **Pro: проверка лицензии** | ✅ нужен |
| OAuth mobile (client_secret) | ✅ нужен |
| Cloud Sync настроек | ✅ нужен |
| Push «загрузка готова» | ✅ нужен |
| Прокси аудиофайлов | ❌ **не делаем** |

**Стек:** FastAPI · PostgreSQL · OAuth2 + JWT · VPS / Railway / Timeweb.

**MVP API:**

```
GET  /health
GET  /v1/version
POST /v1/license/activate      # Pro-ключ
GET  /v1/auth/{provider}/url
GET  /v1/auth/{provider}/callback
POST /v1/sync/settings         # Cloud Sync (Pro)
```

- [ ] Лицензии: генерация, активация, привязка к устройству
- [ ] OAuth proxy для VK / Yandex
- [ ] **Не** хранить и не проксировать музыку

**Оценка:** 1–2 нед. MVP · +2–3 нед. sync.

**💰 Монетизация:** подписка Cloud Sync, валидация Pro-ключей.
